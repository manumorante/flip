M Flip - jQuery Plugin
=============================

A lightweight implementation of the 'flip' effect using CSS styles.

Demo
----

Codepen: http://codepen.io/unavezfui/full/Byryep/
Github: http://manumorante.github.io/flip/

Usage
------

	$('#my-element').mflip();

Dependencies
------------

* jQuery 1.6.0+
