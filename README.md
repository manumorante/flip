M Flip - jQuery Plugin
=============================

A lightweight implementation of the 'flip' effect using CSS styles.

Demo
----

http://codepen.io/unavezfui/full/Byryep/

Usage
------

	$('#my-element-to-watch').mflip();

Dependencies
------------

* jQuery 1.6.0+
